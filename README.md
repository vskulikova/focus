# Фокус

Проект сушки: подход, трекинг, персональные данные. Source of truth для всего, что связано с питанием.

Life-planner (`~/0_Projects/life-planner`) ссылается на этот репо для nutrition-цели и фактов. Hermes (Telegram-buddy на oldmac) читает отсюда цель и пишет сюда лог.

## Структура

- `data/sushka-2026.md` — **план сушки 2026**: цели по неделям, еженедельный трекинг (source of truth для целей и прогресса)
- `data/logs/YYYY-MM.jsonl` — **факт по дням**: append-only лог nutrition (пишет Hermes вечером)
- `data/structured/approach.md` — методология тренера Леры (учебник)
- `data/structured/personal-progress.md` — исторический прогресс по фазам
- `data/raw/` — сырые данные переписок
- `docs/data-model.md` — контракт данных и схема `nutrition_log`
- `docs/hermes-integration.md` — спека интеграции с Hermes

## Источники методологии

- Переписка с Лерой Тренеркой «Белые медведи» (июль–декабрь 2022)

## Разделение ответственности

- **Этот репо (focus)** — методология, целевая фаза, факты по дням.
- **life-planner** — стратегические ритуалы (`/weekly` агрегирует факты отсюда, помогает пересмотреть фазу).
- **Hermes (oldmac)** — операционный слой: утренний бриф цели, вечерний приём отчёта (цифры или скриншот FatSecret).
